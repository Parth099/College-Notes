# The Main Idea:
> **Divisibility Theorem**: 
> Let $a \in \mathbb{Z}^+$
> $$a|b \iff b \text{ mod } a = 0$$
> English: $a$ 'divides' $b$ if and only if the remainder of the division is $\mathbf{0}$

The proof is trivial and just follows the definition of divisibility for *integers*.
